# 🪖 Smart Task Assignment System

A real-time task assignment system that supports **automatic and manual task allocation**, **multi-task handling**, and **workload tracking**.

---

## 🚀 Live Demo

👉 https://task-assignment-system-9uge37voqiyutvfqrkxkpw.streamlit.app/

---

## ✨ Features

### 🔄 Dual Assignment System
- **AUTO Mode** → Automatically assigns tasks based on priority and workload  
- **MANUAL Mode** → Assign multiple tasks to any member manually  

### 👥 Role-Based Access
- **Commander**
  - Add soldiers
  - Create tasks
  - Assign tasks (auto/manual)
  - Monitor workload  

- **Member**
  - View assigned tasks
  - Track task duration
  - Complete tasks
  - View task history  

---

### 📊 Workload Tracking
- See how many tasks each soldier is handling  
- Balanced task distribution  

---

### ⏱️ Time Tracking
- Task start time  
- Task completion time  
- Live running duration  

---

### 📋 Task Management System
- View tasks in:
  - ⏳ Waiting  
  - ⚙️ Active  
  - ✅ Completed  

---
